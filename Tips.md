# Visual Studio Tips

## Edit `Package.appxmanifest` in VS.
`TargetDeviceFamily Name` & `MaxVersion`.

Example:
```XML
<Dependencies>
    <TargetDeviceFamily Name="Windows.Holographic" MinVersion="10.0.10240.0" MaxVersionTested="10.0.10586.0" />
</Dependencies>
```
